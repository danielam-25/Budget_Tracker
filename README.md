This budget tracker is designed to allow you to keep track of your spending and to help you reach ypur financial goals that you can list on the app as well.
I used the ai grok to help me make the application, the description i gave it was:
Create a budget tracker application for easy personal use
Home page should allow users to enter amount that they want to budget
Should also allow users to select from a drop down where the money will be going to 
Should also have a tab where you can add new budgets 
Should all be on one page 
Include bootstrap CDN and the bootstrapwatch.com vapor theme. Include navigation 
All data should be stored in JSON format 
Login is not required
